Frequently Asked Questions

  • What can I use this app for?

    You can use this application to view html of any website. This means that you can checkout how your favourite websites are designed internally and by this learn HTML and CSS. To learn how to use it you please read this article.

  • How do I inspect code of a particular html element?

    For this you have two options. First you can use touch to Inspect feature. All you have to do is to touch finger-tap icon in the top menu and then touch the element that you want to inspect. You can also search in the full website source code.

  • Is it real?

    The code changed on any website will only be stored locally on your device, therefore it will disappear after the page refresh. Therefore it won't affect the real version of the website.
